# Astil Codex UI Prototype

A dependency-free clickable concept for the future Unity desktop client. It validates information architecture, modes, privacy controls, action approvals, activity status, conversation layout, and the visual direction before production UI work begins.

## Open locally

Double-click `index.html` or open it in any modern browser. No server, installation, account, API key, or internet connection is required.

## Interactive elements

- Switch between Companion, Assistant, Focus, Developer, and Creator modes.
- Click suggested prompts or type a message and press Enter.
- Toggle the simulated microphone.
- Cycle through privacy policies.
- Trigger a tool approval by using Developer or Creator mode.
- Approve or deny the simulated task.
- Use the emergency stop.
- Toggle local preference memory.
- Start a new session or use the compact avatar layout.

## Important limitation

This is a visual and interaction prototype—not a connected AI application. The avatar is an original inline SVG placeholder rather than a production VRM model. Responses and actions are simulated, and the prototype cannot access computer files, execute commands, use a microphone, or contact an AI provider.

The final Windows interface is planned for Unity/C# and will communicate with the permission-controlled Astil Codex core through versioned local IPC.
